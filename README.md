# 김과외 자동화 프로그램 베타 안내

# 📥 설치 방법

1. 노트북/PC에서 **Chrome** 브라우저를 사용합니다.
2. **Tampermonkey** 확장 프로그램을 설치합니다.
3. Tampermonkey 아이콘을 클릭한 뒤 **Enable Scripts**가 활성화되어 있는지 확인합니다.
4. 아래 링크에 접속합니다.

   https://github.com/leeeunhyeok1/potential-funicular/raw/refs/heads/main/kimstudy.user.js

5. Tampermonkey 설치 화면으로 자동 이동되면 **Install** 버튼을 눌러 설치합니다.
6. 설치가 완료되면 김과외 사이트에 접속합니다.
7. 제가 보내드린 **라이선스 키**를 입력하면 바로 사용하실 수 있습니다.

---

# 🚀 사용 방법

자동화는 아래 순서로 진행됩니다.

> **학생 수집 → 검토 → 메시지 전송**

## 1. Settings 설정

처음 사용하시는 경우 먼저 **Settings**에서 필요한 정보를 모두 입력해주세요.

입력이 끝나면 **Save** 버튼을 눌러 저장합니다.

---

## 2. 학생 수집

1. 프로그램 페이지에서 **Start New Run** 버튼을 누릅니다.
2. 몇 명의 학생을 수집할지 **Limit** 값을 입력합니다.
   - 예를 들어 **30**을 입력하면 최대 30명의 학생을 수집합니다.
3. **Start** 버튼을 누릅니다.
4. 김과외에서 원하는 조건(지역, 과목, 학년 등)으로 학생을 검색합니다.
5. 학생 목록이 보이는 페이지를 열어두면 화면에 로드되는 학생 정보가 자동으로 수집됩니다.
6. 2페이지, 3페이지 등으로 이동해도 수집은 계속 진행됩니다.
7. 원하는 만큼 학생 정보가 모였다면 **Done** 버튼을 눌러 수집을 종료합니다.
8. 이후 프로그램이 자동으로 **Scraping** 단계를 시작합니다. 잠시 기다려주세요.
9. 좌측 하단의 **Phase**가 **Collecting → Review**로 변경되면 **Review** 탭으로 이동합니다.

---

## 3. 검토 (Review)

Review 탭에서는 수집된 학생 목록을 확인할 수 있습니다.

여기에서

- 메시지를 보낼 학생
- 제외할 학생

을 직접 선택할 수 있습니다.

검토가 끝났다면 **Send** 버튼을 눌러 전송을 시작합니다.

---

## 4. 메시지 전송

전송이 시작되면 프로그램이 학생 페이지를 자동으로 이동하며 작업을 진행합니다.

**전송이 끝날 때까지 브라우저 탭을 닫거나 새로고침하지 말아주세요.**
